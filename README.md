# PokeMemory

A memory card game using Pok√©mon. The goal is to click every card displayed on the screen only once to win. Challenge yourself with 3 difficulty settings and try to succeed!

## Installation

To run the project locally, follow these steps:

1. Clone the repository:

`git clone https://github.com/itsrilay/pokememory`

2. Navigate to the local repository folder:

`cd pokememory`

3. Install dependencies:

`npm install`

## Usage

1. Start the app by running:

`npm run dev`

2. Open a browser and visit [http://localhost:5173](http://localhost:5173).
